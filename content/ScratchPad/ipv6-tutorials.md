# 🧠 IPv6 Tutorial for IPv4 Experts

## 🔰 Start Here: IPv6 for IPv4 Users

- 🎥 [**Fireside Friday – IPv6 For IPv4 Users**](https://www.youtube.com/watch?v=8XReY3Yf6Ho)  
    A practical video that compares IPv6 directly to IPv4 concepts:
    
    - Why IPv6 exists
    - Address allocation differences
    - Multicast vs broadcast
    - Packet filtering and DNS
    - ARP replacement (NDP)
- 📘 [**Hello IPv6: a minimal tutorial for IPv4 users**](https://metebalci.com/blog/hello-ipv6/)  
    A concise blog post that explains:
    
    - No ARP (replaced by NDP)
    - No broadcast (replaced by multicast)
    - NAT is mostly unnecessary
    - SLAAC and DHCPv6 for address assignment  
        Written by someone who set up IPv6 at home using Ubiquiti gear—very relatable.

## 🛠️ Transition Techniques & Dual Stack

- 🌉 [**IPv6 Basic Series — What Is IPv6 over IPv4?**](https://www.youtube.com/watch?v=iUQ5nolG8xo)  
    Learn how IPv6 can coexist with IPv4 using:
    
    - Manual tunnels
    - 6RD and 6to4
    - Dual-stack setups
- 🧭 [**Mastering IP Addressing: Your Guide to IPv4 and IPv6**](https://www.youtube.com/watch?v=0gUrxwrYHQY)  
    Covers routing protocols like OSPF and how dual-stack configurations work.
